# AI Spend Audit

AI Spend Audit is a SaaS-style web application that helps startups optimize their AI tooling costs.

## Features

- AI tool spend analysis
- Cost optimization recommendations
- AI-generated summaries
- Shareable audit URLs
- Email delivery
- Supabase backend
- Responsive UI

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI
- Resend
- Vercel

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=

NEXT_PUBLIC_SUPABASE_ANON_KEY=

OPENAI_API_KEY=

RESEND_API_KEY=
```

## Deployment

Deploy using Vercel.
